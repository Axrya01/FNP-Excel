# Ferns N Petals Sales Analysis Dashboard

This project is an Excel-based sales analysis dashboard created to analyze business performance for Ferns N Petals. The dashboard provides insights into revenue trends, product performance, customer behavior, and order patterns using data visualization techniques.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization
- Interactive Slicers

## Project Overview

The goal of this project is to transform raw sales data into meaningful insights through an interactive Excel dashboard. The dashboard allows users to analyze sales performance across different categories such as product types, occasions, locations, and customer segments.

## Key Insights

- Total revenue generated from sales
- Best performing product categories
- Sales distribution across cities
- Order trends across different months
- Occasion-based purchasing patterns
- Gender-based customer analysis

## Dashboard Features

- Interactive filters using slicers
- Dynamic pivot charts
- KPI metrics for revenue and orders
- Category and gender-based analysis
- City-wise sales visualization
